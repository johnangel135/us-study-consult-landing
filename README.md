# UStart Global Landing Page

Landing page quảng cáo khóa học và tư vấn du học Mỹ.

## Tech
- HTML5
- CSS3 (responsive)

## Chạy local
Chỉ cần mở `index.html` trên trình duyệt.

## Nội dung chính
- Hero section + CTA đăng ký tư vấn
- Khóa học nổi bật (IELTS, SAT, Essay)
- Dịch vụ tư vấn trọn gói
- Kết quả học viên
- FAQ
- Form đăng ký tư vấn

## Tùy chỉnh nhanh
- Thay tên thương hiệu trong `index.html` (`UStart Global`)
- Thay màu trong `styles.css` phần `:root`
- Cập nhật link Open Graph (`og:url`) trước khi deploy
